# Terra Box

## :popcorn: 프로젝트 Front-end/Back-end 소개

`Terra Box` 사이트 클론코딩 프로젝트

대표적인 영화 예매 시스템인 메가박스 사이트를 클론코딩 하였습니다.
영화 예매 및 다양한 필터링 기준을 제공하고 소셜 로그인 가능한 사이트입니다.
